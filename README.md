# AI Tutor Chinese Speech Dataset Portfolio

## 📖 Project Overview
This portfolio is prepared for the **Advanced Candidate** application for xAI's "AI Tutor - Chinese" position, focusing on fine-grained annotation and processing of dialect speech data.

- **Dialect Type**: Xiangxi Fenghuang County dialect (Southwestern Mandarin)
- **Dataset Size**:
  - 🗣️ Dialect speech samples: 12 natural conversation audio clips (16kHz/16bit/mono WAV)
  - 📝 Annotation file: Structured JSON containing dialect transcription, Mandarin translation, and dialectal lexical features
  - 🎤 Mandarin voice sample: 1 high-quality speech sample
  - 🌐 English voice sample: 1 high-quality speech sample

## 🎯 Data Standards & Annotation Methodology
- **Audio Standards**: Following common speech AI standards (16kHz sample rate, 16bit bit depth, mono channel) for recording and processing.
- **Annotation Precision**: Each item has been **manually refined** to ensure accuracy of dialect transcription, Mandarin translation, and dialectal lexical annotation, demonstrating deep attention to semantic details.
- **Quality Control**: All audio recorded in a quiet indoor environment with high SNR and no background noise.
